# task-project

Адаптивный лендинг жилого комплекса, реализованный по дизайн-макету в соответсвии с ТЗ.  
Проект демонстрирует компонентный подход, адаптивную вёрстку, работу с модальными окнами, формами, маской телефона и пользовательскими интерактивными элементами.
